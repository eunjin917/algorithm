## 시간 복잡도
- 문제의 크기(N)에 따른 시간이 얼마나 걸리는지 `최악의 경우` 예측 가능
- `Big O Notation`
  - ex. O(1), O(N), O(N<sup>2</sup>), O(NlogN), O(N!)
  - 99%의 경우에 0.01초 + 1%의 경우에 100초 걸리는 알고리즘 => <100초>
- 필요 이유
  - 문제를 풀기 전에 먼저 생각한 방법의 시간 복잡도를 계산해보고, 이게 시간 안에 수행될 가능성이 있는지 확인 가능
- `N이 1억 = 약 1초 소요`
- 계산 방법
   1. 상수는 버림
       - O(3N<sup>2</sup>) = O(N<sup>2</sup>)
       - O(5) = O(1)
   2. 2개 이상 항 존재 경우, 변수가 같으면 큰 거 빼고 다 버림
       - O(N<sup>2</sup> + NlogN) = O(N<sup>2</sup>)
- 예제
  - 총 N명의 사람이 식당에 방문
  - 식당에는 메뉴 M개, 메뉴판 1개
  - 사람 1명이 메뉴판 읽는데 걸리는 시간 : `O(M)`
  -  주문한 모든 메뉴는 동시에 나왔고, 각 사람 i가 식사를 하는데 걸리는 시간 : A<sub>i</sub>
  -  각 사람이 계산을 하는데 걸리는 시간 : O(P)
  -  각 사람이 메뉴판에 있는 모든 메뉴를 읽는 시간 복잡도 : `O(NM)`
  -  모든 사람이 식사를 마치는데 걸리는 시간 : `O(max(A`<sub>i</sub>`))`
  -  식사를 모두 마친 다음 한 줄로 서서 각자 계산을 하는 시간 복잡도 : `O(NP)`
- 예제 : 1~N 합 계산
  - O(N)
    ```C
    int sum = 0;
    for (int i=1; i<=N; i++) {
      sum += i;
    }
    ```
  - O(N<sup>2</sup>)
    ```C
    int sum = 0;
    for (int i=1; i<=N; i++) {
        for (int j=1; j<=N; j++) {
            if (i == j) {
                sum += j;
            }
        }
    }
    ```
  - O(1)
    ```C
    int sum = 0;
    sum = N*(N+1)/2;
    ```

## 메모리
- 보통 가장 많은 공간 차지 : `배열`
- `시간 제한 ~> 메모리 제한`
- 불필요한 공간 없도록 하면 메모리 제한은 알아서 지켜짐

## 입출력
- C++
  - `scanf/printf`
    - cin/cout은 느리기 때문에 입/출력이 많은 문제에서 사용
  - `cin/cout`
    - scanf/printf와 혼용 X
    - 아래 3줄을 추가하면 scanf/printf만큼 빨라짐
        ```C++
        ios_base::sync_with_stdio(false);
        cin.tie(nullptr);
        cout.tie(nullptr);  // 생략가능
        ```
  - endl보다 `\n` 사용

- Java
  - 입력
    ```C++
    Scanner sc = new Scanner(System.in);
    ```
  - 출력
    ```c++
    System.out
    ```
  - 입력이 많은 경우
    ```C++
    BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
    ```
  - 출력이 많은 경우
    1. `StringBuilder` 사용하여 한 문자열로 만든 뒤 출력을 한 번만 사용
    2. `BufferedWriter` 사용
- 입력이 몇 개인지 주어지지 않은 경우
  - `EOF`까지 입력받기
  - C
    ```C
    while (scanf("%d %d", &a, &b) == 2)
  - C++
    ```C++
    while (cin >> a >> b)
    ```
  - Java
    ```Java
    while (sc.hasNextInt())
